# xidian-login

针对西电校园网的自动登录脚本。

## how-to-use

1. 设置校园网账号密码；

2. 安装chrome驱动，并配置驱动位置；

3. pyinstaller打包为可执行程序；

4. 设置自动。

   - bios设置断电重启；

   - Windows10设置任务计划程序。

**注意关闭代理*
